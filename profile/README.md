## 📝 프로젝트 설명

‘STAGE’는 실시간 트래픽 제어 기술을 기반으로, 공정하고 안정적인 공연 예매 환경을 제공하는 티켓 예매 서비스입니다.

STAGE는 공연 오픈과 같이 접속자가 급증하는 상황에서도
모든 사용자가 공평한 기회를 얻고,
좌석 선택 과정에서의 충돌이나 오류 없이 예매를 진행할 수 있도록 설계되었습니다.

특히, 대기열 시스템과 실시간 좌석 선점 기능을 통해
동시 접속 환경에서 발생할 수 있는 중복 예약, 서버 과부하, 결제 충돌 문제를 예방합니다.

</br>

**1. 실시간 좌석 선점 기능**

공연 예매 과정에서 사용자가 특정 좌석을 선택하고 결제를 시도하면, 해당 좌석은 즉시 ‘임시 선점 상태’로 전환되어 다른 사용자 화면에서는 선택이 불가능하도록 비활성화됩니다.

사용자가 결제를 완료하면 좌석은 최종 확정되며, 결제를 진행하지 않고 일정 시간이 경과할 경우 자동으로 선점이 해제되어 해당 좌석은 다시 선택 가능한 상태로 전환됩니다.

</br>

**2. 대기열 기반 공정 입장 시스템**

공연 오픈 후 서버 과부하 등을 방지하기 위해, 좌석 선택 페이지에 접속하는 시점에 현재 활성 사용자 수를 확인하고, 설정된 최대 동시 접속 인원을 초과할 경우 자동으로 대기열에 배치됩니다.

일정 주기로 대기 순번이 갱신되어 차례가 되면 자동으로 좌석 선택 페이지로 입장하게 됩니다.

</br>

**3. 예매 내역 및 상태 관리 기능**

사용자는 마이페이지를 통해 자신이 예매한 공연의 정보를 한눈에 확인할 수 있습니다.
